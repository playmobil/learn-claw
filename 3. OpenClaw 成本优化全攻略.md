## 3. 详细操作步骤

### 3.1 第一阶段：部署 QMD（Query Markdown）本地索引

**逻辑分析：** 传统模式是“填鸭式全量输入”，无论相关与否都把整个知识库塞给 AI。QMD 则是**“精准语义检索”**：它在本地建立索引，仅在对话时提取最相关的片段注入。**重点：** 索引刷新（Update）和向量化计算（Vectorization）全在本地完成，不消耗一分钱云端 Token。

**具体步骤：**

1. **安装 QMD：** 在终端运行安装命令（或通过 OpenClaw 辅助安装）：
2. **刷新索引：** 执行 `update` 指令，扫描文件的增删改，更新分段路径。
3. **向量化更新：** 将片段转化为向量存入本地数据库，用于后续相似度匹配。
4. **配置生效：** 打开 `openclaw.json`，确保 `memory` 字段指向 QMD 模式。
5. **重启网关：** 刷新 OpenClaw 使配置生效。

### 3.2 第二阶段：配置本地模型处理“心跳 (Heartbeat)”

**逻辑分析：** “心跳”是 OpenClaw 的**定时唤醒触发行为**，类似于“监工”。它每隔一段时间检查 `Heartbeat.md` 里的清单，确保长期任务不因为 AI 偶尔的“歇菜”而中断。 心跳是高频且长上下文的行为。由于心跳时需要读取 `memory.md` 和 `agent.md`（这两个文件往往极大），即使输出只有一句“OK”，输入成本也极其惊人。按每 30 分钟一次计算，每月会产生 1,440 次完整的 API 回合扣费。这类“低智商触发任务”完全应交给本地模型。

**具体步骤：**

1. **启动 Ollama：** 确保后台有 Qwen 等模型待命。
2. **修改设置：** 在 OpenClaw 配置中，将 `Heartbeat` 的执行模型指定为本地 `Ollama` 模型。
3. **效果：** 复杂的创作交给云端，机械的“监工”交给本地，省去每月上千次无效 API 支出。

### 3.3 第三阶段：优化订阅与模型选择策略

**逻辑分析：** 虽然 API 计费灵活，但在 OpenClaw 这种高频调用的场景下，订阅制的经济性呈压倒性优势。

- **重要内幕：** 目前 **OpenAI 已收购 OpenClaw**，因此两者的订阅兼容性极佳，处于开放支持状态。
- **禁区警告：** **Anthropic** 和 **Google** 的订阅严禁用于此类外部自动化调用，强行尝试可能导致封号。

**操作指引：**

1. **优先 OpenAI 订阅：** 相比 API 消耗，订阅制能极大缓解账单压力。
2. **成本量化对比：** 谨慎追求最新模型。根据实测，使用 Minimax 的 API 成本约为 30/天，而若切换到 Anthropic 最新的 Opus 4.6 模型，处理同样任务的账单可能飙升至 **500/天**。

### 3.4 第四阶段：执行定制化“成本体检”与流程优化

**逻辑分析：** 高消耗往往源于流程配置的不合理。通过 AI 的自我审计，可以精准打击“隐形浪费”。

**具体步骤：**

1. **发出指令：** 命令 OpenClaw：“请生成一份成本体检报告，列出消耗 Token 最多的 TOP 5 驱动清单，按百分比由大到小排序。”
2. **识别浪费：**
   - **轮询 vs 事件驱动：** 检查是否有些任务在没必要的情况下使用了“每 10 分钟检查一次”的**高频轮询制**。
   - **上下文冗余：** 识别哪些轻量任务携带了不必要的巨量背景资料。
3. **对症下药：** 针对高消耗项，与 OpenClaw 讨论：是改用 QMD 模式、减少轮询频率，还是更换更便宜的小模型。

## 4. 高阶技巧与避坑指南

为了达到极致的边际成本优化，请精细化调整 `openclaw.json`：

**QMD 预算阀门配置：** 在 `limit` 参数下，你可以精准锁死支出上限：

```json
"limit": {
  "maxResults": 5,          // 限制单次注入的最相关的片段数量
  "maxSnippetChars": 500,   // 限制每个片段的最大字符长度
  "maxInjectedChars": 2000  // 总预算阀门：每轮对话注入的总字符上限
}
```

**其他专家建议：**

- **心跳频率：** 建议设置为 **30 分钟**。频率过高会无谓消耗本地资源，过低则可能导致 agent 失去任务一致性。
- **性能调优：** 若本地模型响应极慢，请检查本地内存是否被占满，建议关闭非必要的后台程序，或降级使用参数更小的模型（如 Qwen 1.5B/7B）。
